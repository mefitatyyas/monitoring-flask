# monitoring-flask

### Instalasi

- Clone repository

```
git clone https://github.com/mefitatyyas/monitoring-flask.git
```

- Instal Requirement 

```
$ pip install -r requirements.txt
```

### Push Github

- Add File

```
git add .
```

- Memberi Comment

```
git commit -m "pesan"
```

- Mengirim ke Github

```
git push -u origin main
```

### Pull Github

- Mengambil file dari Github

```
git pull
```
